# Breast Cancer Detection using Logistic Regression

This project predicts whether a breast tumor is **benign** or **malignant** using the **Breast Cancer Wisconsin (Original) Dataset**.  
The model is built using **Python**, **Google Colab**, and **Scikit-Learn**.

---

## Dataset
- Source: UCI Machine Learning Repository  
- Total samples: 699  
- Features describe cell characteristics from fine-needle aspirates  
- Target classes:
  - **2 = Benign**
  - **4 = Malignant**

---

## What the Notebook Includes
- Loading and cleaning the dataset  
- Splitting data into training and testing sets   
- Training a Logistic Regression model  
- Making predictions  
- Evaluating accuracy and confusion matrix
